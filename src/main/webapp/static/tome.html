On gateways

  1. Gateways (TrueCrypt volumes shared on the network) are used to proffer (share) artifacts (files) to other cultists (users), or to clone (copy) artifacts that were proffered by others.
  2. In this way no artifacts touch disk in plain-text.

On creating a gateway

  1. Create a TrueCrypt 7 encrypted volume in that share, using the "create volume" wizard following the instructions below:
    1. Create an encrypted file container
    2. Standard TrueCrypt volume
    3. Volume location should be in the network-share created earlier, with any name
    4. Default encryption options
    5. Volume size should be fixed (as variable sized volumes can cause corruption), and recommend >10GB size if possible (it will fill fast)
    6. I will not store files larger than 4GB on the volume. (Only on Ubuntu as this enables selection of FAT in the next step).
    7. FAT file system
    8. Format
  2. Ensure the file has read-write permissions for all.
  3. Add a gateway for this share, choosing the sink mode for a gateway to receive clones or source mode to proffer artifacts for other cultists to clone.
