Setup a sink:

  1. Create a network-share on your computer with read-write permissions for all.
  2. Create a TrueCrypt 7 encrypted volume in that share, using the "create volume" wizard following the instructions below:
    1. Create an encrypted file container
    2. Standard TrueCrypt volue
    3. Volume location should be in the network-share created earlier, with any name
    4. Default encryption options
    5. Volume size should be fixed (as variable sized volumes can cause corruption)
    6. I will not store files larger than 4GB on the volume. (Only on Ubuntu as this enables selection of FAT in the next step).
    7. FAT files system
    8. Format
  3. Ensure the file has read-write permissions for all
  4. Add a gateway for this share.
